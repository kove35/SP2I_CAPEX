# Logistique import Afrique

Le contexte principal est Chine -> Pointe-Noire, Congo-Brazzaville.

## Points sensibles

```text
consolidation fournisseurs
port de depart Chine
transport maritime
Port autonome de Pointe-Noire
douane Congo
transport local
stockage chantier
sequençage livraison
```

Un gain achat peut etre absorbe par un mauvais remplissage container, une
congestion portuaire, un stockage trop long ou une livraison trop tardive.

## Objectif SP2I

SP2I doit rendre ces risques visibles avant la decision finale :

```text
IMPORT rentable
mais delai et stockage a securiser
```
