# Strategie import SP2I

SP2I doit distinguer trois cas.

## Import recommande

```text
gain financier fort
risque faible ou moyen
delai acceptable
MOQ adapte
cashflow soutenable
```

## Import possible sous controle

```text
gain interessant
mais risque fournisseur, delai ou MOQ a surveiller
```

Ce cas doit etre documente et suivi dans Power BI.

## Local recommande

```text
gain faible
ou risque fort
ou chantier critique
ou cashflow dangereux
ou MOQ disproportionne
```

Le local n'est pas un echec d'optimisation. C'est parfois la decision la plus
fiable pour securiser le chantier.
