# Design system SP2I

Le design cible combine :

```text
Power BI analytics
SAP Analytics
Procore construction
Linear UX
Stripe SaaS premium
Flexport supply chain en support
```

## Principes

```text
dark mode premium
cartes analytiques denses
sidebar enterprise
topbar operationnelle
tableaux lisibles
KPI immediatement visibles
hierarchie orientee pilotage immobilier
```

## Composants UI

```text
KpiCard
AnalyticsCard
AlertCenter
ProjectSelector
Skeleton
ContainerTable
ShipmentTable
ScenarioComparison
SimulationToolbar
```

## Couleurs

Le cockpit utilise un fond sombre bleu/noir, des accents verts decisionnels et
des alertes orange. Les pages DQE historiques conservent leurs styles pour ne
pas casser l'existant.
